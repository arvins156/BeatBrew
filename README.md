### Requires PHP
### `cd spotify-lyrics-api`
### Requires Spotify Cookie/Token
### `export SP_DC=[token here and remove the square brackets]`
### `php -S localhost:8000 api/index.php`
### API from: https://github.com/akashrchandran/spotify-lyrics-api 

### cd to JohnKaraoke
### `npm install`
### `npm start`
